Overview

This Virtual Drawing application utilizes OpenCV and Mediapipe to create an interactive drawing interface controlled by hand gestures. The application uses a webcam to capture hand movements and translates these into drawing actions on the screen. Users can select different drawing tools, colors, and shapes using hand gestures, and can also save their artwork.

Algorithms and Technologies Used

1.OpenCV: A powerful library for computer vision tasks, used here for capturing video, drawing shapes, and handling image processing.

2.Mediapipe: A library by Google that provides machine learning solutions for live and streaming media, used here for hand landmark detection and tracking.
