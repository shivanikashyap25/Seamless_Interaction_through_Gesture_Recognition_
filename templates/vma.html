Virtual Mouse
With AI Assistance

Overview

The Virtual Mouse application integrates hand gesture recognition and voice commands to offer a seamless, touch-free way to control a computer mouse. Using a webcam, the system captures hand movements and interprets them into mouse actions such as moving the cursor, clicking, dragging, and scrolling. Along with gesture control, voice assistance is included for performing system-level commands, checking system status, and managing media, enhancing user interaction and productivity.

Algorithms and Technologies Used

OpenCV: Employed for real-time video input and processing, capturing the hand gestures and translating them into mouse actions.

Mediapipe: Google's machine learning-based framework, used for detecting and tracking hand landmarks, allowing for precise and intuitive cursor control.

Voice Assistance: Voice commands are integrated using services like Google Cloud Speech-to-Text, enabling hands-free interaction for performing various system and media tasks.
