Steps to Use the Application

1.Launch the Application:

: Run the program, and the webcam will initialize to begin detecting hand gestures.

2. Cursor Movement:

Move your hand in front of the webcam to control the cursor. The index finger is used as the pointer for precise movement.

3.Mouse Actions:

Perform left-clicks, right-clicks, dragging, and scrolling by using the corresponding gestures.

4.Voice Commands:

Issue voice commands for system controls such as opening apps, checking battery status, controlling volume, or getting the current date and time.

5.Special Gestures:

Use the "all fingers raised" gesture to execute an action (equals gesture) and the "thumb and little finger raised" gesture to reset the action (clear gesture).

Exit Application

Press ctrl to exit the application when you are finished using the virtual mouse.
