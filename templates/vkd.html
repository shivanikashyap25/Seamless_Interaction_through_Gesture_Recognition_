Steps to Use the Virtual Keyboard Application

1.Launch the Application:

Start the program, which initializes the webcam and activates the virtual keyboard.

2.Color Detection for Enhanced Tracking:

Use a green object or your hand to interact with the virtual keys, leveraging the enhanced tracking for smoother performance.

3.Select Keys:

Hover your finger over a virtual key and use the pinching gesture or similar hand gesture to type the key into the active Notepad window.

4.Change Font Settings:

Modify font size, color, and style by using predefined gestures or voice commands.

5.Delete and Formatting:

Perform gestures for deleting text, inserting spaces, or toggling formatting like bold and italic.

6.Voice Commands:

Use voice commands to type characters, change font settings, or execute system commands such as saving the file or opening Notepad.

Exit Application

Press CTRL+C to exit the virtual keyboard application.
