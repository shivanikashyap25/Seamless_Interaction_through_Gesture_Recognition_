How it Recognizes Hand Gestures

Hand Landmark Detection: Using Mediapipe's model, the system tracks 21 hand landmarks in real-time, mapping them to specific mouse functions.

Gesture Recognition: Specific hand gestures are recognized based on the arrangement and movement of hand landmarks, triggering actions like left-click, right-click, drag, and scroll.

Mouse Actions and Corresponding Gestures

Move Cursor: The cursor follows the motion of the user's index finger, acting as the pointer.

Left Click: A pinch gesture, where the thumb and index finger come together, simulates a left-click action.

Right Click: A closed fist gesture signals a right-click function.

Drag and Drop: Perform a drag by pinching with the thumb and index finger and holding the gesture while moving the hand. Release the pinch to drop.

Scroll: Scroll up or down by moving the hand vertically while holding an open palm gesture.

Voice Assistance for System Control

The Voice Assistance feature enhances the virtual mouse experience by allowing users to perform various system-level tasks using voice commands. This includes opening apps, controlling media, and checking system information:

Opening Applications:

Users can open programs like browsers, media players, or file explorers by saying, "Open Chrome," "Open File Explorer," or similar commands.

Status Commands:

o Battery Check:

Ask the system, "Check battery status," to get the current battery percentage and whether the device is charging.

o Volume Level:

Control the volume by saying commands like, "Increase volume," "Decrease volume," or "Mute."

o Date and Time:

Get the current date, time, and day by simply asking, "What's the time?" or "What's today's date?"

o Media Control:

-- Play/Pause Music:

Start or stop playing music with commands like, "Play music," or "Pause song."

-- Next/Previous Song:

Navigate through your playlist by saying, "Next song," or "Previous song."

o Browser Control:

-- Open New Tab:

Open a new browser tab by saying, "Open new tab."

These voice commands provide a hands-free way to manage both system and media functions, further enhancing the user experience.

Gesture for Special Functions

Equals Gesture: To execute an action (such as confirming a selection or running a command), the user raises all fingers, signaling the system to complete the task.

Clear Gesture: To reset or cancel an input or action, the user raises only the thumb and little finger. This predefined gesture serves as the clear command, allowing the user to back out of the current task or reset the input.
