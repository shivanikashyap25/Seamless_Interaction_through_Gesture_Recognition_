Steps to Use the Application

1.Launch the Application:

Start the calculator by running the script that activates the webcam and initializes the gesture-based interface.

2.Input Numbers:

Use hand gestures corresponding to numbers (from 0 to 9) to input values into the calculator.

3.Select Operations:

Choose from basic arithmetic or advanced operations like trigonometric functions, logarithms, or integrals by performing the appropriate gesture.

4.Perform Calculations:

After inputting numbers and selecting an operation, perform the gesture for the equals (=) function to execute the calculation and display the result.

5.Memory Functions:

Store and recall values in memory with specific gestures designed for memory addition, subtraction, and clearing.

6.Clear Input:

Reset the calculator by performing a clear gesture to start a new calculation.

7.Exit Application

Press CTRL +C in the terminal to exit the application when finished.
