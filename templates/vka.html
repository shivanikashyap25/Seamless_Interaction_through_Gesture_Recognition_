Overview

The Virtual Keyboard application offers an intuitive and hands-free way to type on a virtual keyboard using hand gestures. This system is particularly useful in environments where touch-based interactions may be inconvenient. Utilizing a webcam, the application tracks hand gestures to perform typing actions on an active text window like Notepad. With the integration of color detection to enhance gesture tracking, especially with green objects, and advanced features like font size, font color, and other text-related settings, the application provides a full-fledged virtual typing experience which we planned to implement in stage-3.

Algorithms and Technologies Used

OpenCV: Used for real-time video input, processing, and detecting hand movements. OpenCV also handles image processing and key detections related to gestures.

Mediapipe: A machine learning framework by Google that provides efficient and accurate hand landmark detection and tracking, enabling precise gesture control for typing.

Color Detection: For optimized performance, the application detects green objects faster, enhancing the accuracy and speed of gesture recognition.

Voice Assistance: Voice commands add an extra layer of control for typing, font customization, and application management.
