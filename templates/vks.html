Gesture-Based Keyboard Actions

Type a Key: The user hovers their hand over a virtual key mapped on the screen, and by making a specific gesture (e.g., pinching or tapping gestures), the corresponding key is typed into the active text window such as Notepad.

Delete a Character: The system recognizes a swiping gesture (e.g., left-to-right) to delete the last typed character.

Space: A unique gesture, like tapping two fingers, adds a space between words.

Enter: Raising all five fingers and closing them into a fist will signal the Enter key function.

Color Detection and Enhanced Tracking

Optimization: In this stage of implementation, the system improves gesture recognition by focusing on green-colored objects, which are detected faster than other colors. This is especially useful for selecting keys and performing other typing actions with increased accuracy and speed.

Hand Tracking: Mediapipe's hand landmark detection maps the hand's position to a virtual keyboard layout, allowing users to "press" keys by hovering their fingers over specific areas.

Customization Features for Font

Font Size Control: Users can change the font size of the text in Notepad using gestures or voice commands. For instance, raising the hand vertically with fingers spread wide will increase the font size, while closing the hand into a fist will decrease the font size.

Font Color Control: To change the font color, users can use a circular motion gesture or simply voice a command like "Change font color to blue." Available colors include: o Black o Red o Blue o Green o Yellow o White o Cyan o Magenta

Bold/Italic/Underline: A specific gesture (such as two-finger pinching) or a voice command can toggle between text styles like bold, italic, or underline.
